Instructions to Cube Transformation Program
cube_transformations_demo.py by 

How to use (all operations):

    Toggles:
    F - toggles between solid cube and wireframe cube
    R - Reset cube

    Rortaion:
    Arrow Keys(up and down) - rotate with respect to x axis
    Arrow Keys(left and right) - ratate with respect to y axis
    Left shift + Arrow Keys - rotations above + rotate with respect to z axis

    Tranlastion:
    W - move up (+y)
    A - move left (-x)
    S - move down (-y)
    D - move right (+x)
    Hold left shift + W - move forward (+z)
    Hold left shift = S - move backward (-z)

    Stretching
    x + '=' (Plus/Equals key) or '-' (Minus key) - Strech cube on x axis
    y + '=' (Plus/Equals key) or '-' (Minus key) - Strech cube on y axis
    z + '=' (Plus/Equals key) or '-' (Minus key) - Strech cube on z axis

    Scaling
    = (Plus/Equals key) - Scale cube up/make bigger 
    '-' (Minus key) - Scale cube down/make smaller
